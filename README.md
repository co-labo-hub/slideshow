# slideshow

Github Acitons 学習用  

下記設定でスライドショーを作成します。  

### 設定
・画像ファイル：[数字].jpg  
・BGMファイル：bgm.mp3  
・画像は5秒ごと、よってBGMはファイル数×5秒のものを用意する  
・出力ファイル：out.mp4  

### 出力例
https://user-images.githubusercontent.com/66151500/178160034-08b75e6e-308e-4f62-be5f-ed9606b6e528.mp4
